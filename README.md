# 📒 Contact Manager Console (C#)

## 📌 Project Overview
This is a **Console-based Contact Manager application** developed in C#.  
It provides a simple menu-driven interface to manage contacts, including adding, editing, searching, deleting, listing, and clearing contact data.

The project demonstrates practical usage of **Object-Oriented Programming (OOP)** principles and separation of concerns using interfaces.

---

## ⚙️ Features

✔️ Add new contact  
✔️ Edit existing contact  
✔️ Find contact by name  
✔️ Remove contact (by name + phone number)  
✔️ Display all contacts  
✔️ Remove contact by index  
✔️ Clear all contacts  
✔️ Display total number of contacts  
✔️ Simple console menu system
